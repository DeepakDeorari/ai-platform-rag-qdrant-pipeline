Kubernetes is an open source container orchestration platform.

Pods are the smallest deployable units in Kubernetes.

Pods run on worker nodes.

The scheduler decides where pods should run.

A deployment manages pod replicas.

A service exposes pods.

An ingress provides HTTP routing.

ConfigMaps store configuration.

Secrets store sensitive information.

StatefulSets manage stateful workloads.

DaemonSets run one pod per node.

Jobs execute batch workloads.

Horizontal Pod Autoscaler scales workloads.

Persistent Volumes provide storage.

Namespaces provide isolation.

RBAC controls access.

Kubelet runs on every worker node.

The API Server is the control plane entry point.

etcd stores cluster state.

Controllers reconcile desired state.
